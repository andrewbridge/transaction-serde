Function toMs

Converts time components to milliseconds since midnight.

This is the inverse of toTimeComponents. All parameters default to 0.

Example

toMs(14, 30);
// => 52200000

Example

toMs();
// => 0

Example

toMs(23, 59, 59, 999);
// => 86399999
